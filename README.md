# Glaze HPP Analyzer
Introduction:
Glaze adalah adalah alat analyst yang dirancang untuk membantu UMKM dalam menghitung HPP menggunakan metode Full Costing.

Latar Belakang:
Banyak UMKM di Indonesia yang belum memahami penetapan harga pokok produksi sehingga sering mengambil prive secara berlebihan dan mengganggu modal kerja. Glaze hadir untuk menjadi solusi sederhana bagi permasalahan tersebut.

Fitur:
1. Form Input Biaya: input biaya bahan baku, tenaga kerja, dan overhead yang sudah terstruktur.
2. Perhitungan Full Costing: menghitung total biaya produksi, HPP per unit, dan laba per unit.
3. Dashboard Analisis: menampilkan biaya tertinggi beserta nominalnya untuk membantu efisiensi.
4. Analisis Laba: menghitung laba kotor, laba bersih, dan rekomendasi harga jual.
5. Batas Aman Prive: rekomendasi nominal maksimal yang boleh diambil oleh pemilik usaha.
